Modaclouds-SDA
==============
## How to run

```
sudo ./run_main.sh /usr/local/MATLAB/MATLAB_Compiler_Runtime/v81/ mode port
```
where mode is either "kb" or "file" and port is the listerning port to receive data from the MODAClouds DDA.
